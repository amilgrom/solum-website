# Solum Real Estate Group Website

## What This Is

A professional website for Solum Real Estate Group LLC (solumreg.com), a new commercial real estate company founded by Andrew Milgrom and Geoff Hanson. The site establishes credibility with property owners, brokers, and lenders while clearly articulating the company's investment approach and market focus. Hosted on GitHub Pages as a static site.

## Core Value

**Establish credibility with lenders.** Everything else serves this. If lenders see a professional, capable team with a clear investment thesis and proper structure, the rest follows.
