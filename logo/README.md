# Logo Assets

Source logo files for Solum Real Estate Group.

## What's deployed

`public/logo.png` — a copy of **SREG Logo - Transparent Background (Small Margins).png** from this folder. This is the version used on the website (transparent background, tight crop, olive green mark on cream).

If the logo is ever updated, replace `public/logo.png` with the new transparent-background version.

## Files in this folder

| File | Notes |
|------|-------|
| SREG Logo - Transparent Background (Small Margins).png | **Source for public/logo.png** — tight crop, use on light backgrounds |
| SREG Logo - Transparent Background.png | Same logo, wider margins |
| SREG Logo - White Background.png | Baked white background |
| SREG Logo - Black Background.png | Light rendering on black — for dark backgrounds |
| SREG Logo - Background Color Options.png | Brand-approved background color combinations |
| Color Codes.png | Official brand color palette with hex codes |

## Brand colors (from Color Codes.png)

- Primary logo green: `#7F8A6A`
- Primary light background: `#F2EADF` (used as site background)
- Approved dark backgrounds: `#121212`, `#1E2A22`, `#342F2A`, `#252230`
